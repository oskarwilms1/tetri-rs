# Tetri-rs

Tetri-rs is a simple Tetris clone written in **Rust** using the **Bevy** game engine.  

**Disclaimer:** This project is a fan-made implementation for learning and entertainment purposes. I do **not** claim ownership of the original Tetris concept.

## Controls

- **A/ArrowLeft** – Move left  
- **D/ArrorRight** – Move right  
- **S/ArrowDown** – Move down  
- **W/ArrowUp** – Rotate piece  
- **Space** - Drop piece
